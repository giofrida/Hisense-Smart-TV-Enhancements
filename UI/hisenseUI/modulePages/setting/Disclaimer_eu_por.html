Minimização de dados

Tomamos todos os passos razoáveis para garantir que os seus Dados Pessoais que Processamos sejam limitados aos Dados Pessoais razoavelmente necessários em conexão com os propósitos estabelecidos nesta Política de Proteção de Dados.

Retenção de dados

Nós tomamos todos os passos razoáveis para garantir que seus Dados Pessoais sejam processados apenas pelo período mínimo necessário para os fins estabelecidos nesta Política de Proteção de Dados.

Os critérios para determinar a duração para a qual manteremos seus Dados Pessoais são os seguintes: reteremos cópias de seus Dados Pessoais em um formulário que permita a identificação apenas durante o tempo necessário em conexão com os objetivos estabelecidos nesta Proteção de Dados Política, a menos que a lei aplicável exige um período de retenção mais longo. Em particular, podemos manter seus Dados Pessoais durante o período de tempo necessário para estabelecer, exercer ou defender quaisquer direitos legais.

• O direito de não fornecer seus Dados Pessoais para nós (no entanto, note que talvez não possamos fornecer-lhe o benefício total de quaisquer Sites, Aplicativos e Serviços Inteligentes que você solicite, se você não nos fornecer seus Dados Pessoais - Por exemplo, talvez não possamos processar seus pedidos sem os detalhes necessários);

• o direito de solicitar acesso ou cópias de seus Dados Pessoais, juntamente com informações sobre a natureza, Processamento e divulgação desses Dados Pessoais;

• o direito de solicitar a rectificação de quaisquer imprecisões nos seus Dados Pessoais;

• o direito de solicitar, por motivos legítimos:

o o remoção de seus dados pessoais; ou

o o restrição do processamento de seus dados pessoais;

• Onde processamos seus dados pessoais com base no seu consentimento, o direito de retirar esse consentimento (observando que tal retirada não afeta a legalidade de qualquer processamento realizado antes da data em que recebemos aviso de tal retirada e não impedir o processamento de seus dados pessoais com base em outras bases legais disponíveis); e
